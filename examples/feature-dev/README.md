# Feature Dev

OIAP rewrite of Anthropic's Claude Code `feature-dev` plugin.

Source plugin: <https://github.com/anthropics/claude-code/tree/main/plugins/feature-dev>

This example is for demonstrative purposes only. The upstream repository license
permits this adapted demonstration.

The OIAP version maps the original guided feature workflow to an invocation,
command prompt, and three custom agent definitions for exploration,
architecture, and review.
