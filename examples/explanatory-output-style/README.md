# Explanatory Output Style

OIAP rewrite of Anthropic's Claude Code `explanatory-output-style` plugin.

Source plugin: <https://github.com/anthropics/claude-code/tree/main/plugins/explanatory-output-style>

This example is for demonstrative purposes only. The upstream repository license
permits this adapted demonstration.

The OIAP version expresses the original SessionStart behavior as a portable
`session_start` hook that injects concise educational guidance at the start of a
session.
